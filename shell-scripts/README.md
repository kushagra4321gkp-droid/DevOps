# Shell Scripts Collection

This directory contains **basic shell scripts** written for learning and practicing Linux and DevOps fundamentals. These scripts cover common concepts like conditionals, loops, system checks, and simple AWS-related utilities.

The goal of this folder is to build a strong foundation in **Bash scripting**, which is essential for DevOps and system administration roles.

## 📁 Scripts Overview

| Script Name               | Description                                                  |
| ------------------------- | ------------------------------------------------------------ |
| `aws_resource_tracker.sh` | Tracks AWS resources (basic usage, learning-oriented)        |
| `commandsList.sh`         | Contains commonly used Linux commands for reference          |
| `forLoop.sh`              | Demonstrates usage of `for` loops in shell scripting         |
| `ifelse.sh`               | Demonstrates `if-else` conditional statements                |
| `nodeHealth.sh`           | Checks basic system or node health (CPU, memory, disk, etc.) |
| `test.sh`                 | Simple test script used for practice                         |

## ▶️ How to Run a Script

1. Give execute permission:

```bash
chmod +x script_name.sh
```

2. Run the script:

```bash
./script_name.sh
```

Or using bash:

```bash
bash script_name.sh
```
